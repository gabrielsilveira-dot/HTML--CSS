Inserindo Videos localmente

Este video foi inserido localemnte, ou seja no meu computador.

Para colocar um video utiliza a TAG video , dentro da TAG coloca source src para escolher o video e o formato

O parametro colocado em <video ....... é o width para o tamanho, controls. para colocar controles no video, e poster para incluir uma capa no video

Escolhe o type igual em audio, imagens etc.
